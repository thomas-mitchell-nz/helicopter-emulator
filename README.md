# helicopter-emulator
A TIVA-Orbit board was used to interact with and control a separate microcontroller-based helicopter emulator. The purpose of the real-time control software was to adjust the altitude and yaw of a simulated helicopter. This was achieved by executing control tasks on the FreeRTOS operating system, in response to commands from a user interface. Stable flight was attained using a PID algorithm. 

This project was created using Code-Composer Studio (CCS). Refer to the 'Setup Tutorial' document for instructions on setting up a workspace.
